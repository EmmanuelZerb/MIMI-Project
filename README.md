# MIMI - Projet Blender

Projet 3D Blender partage entre Emmanuel et sa copine.

## Structure

- `MIMI.blend` - Fichier principal Blender
- `renders/` - Dossier pour les rendus (a creer si besoin)

- `main` - Version stable

- `feature/nom` - Nouvelles fonctionnalites
